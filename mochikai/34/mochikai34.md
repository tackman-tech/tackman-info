# Building Blocks of Tabletop Game Design 紹介

## ボードゲームのメカニクスパターン集

例えば「経済(Economics)」のカテゴリでは、

```
"Exchanging" "Trading" "Market" "Loans" "Upgrades
```

のような粒度で紹介されている

### 例

- キング・オブ・トーキョー
